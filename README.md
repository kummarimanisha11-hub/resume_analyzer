# resume_analyzer
Resume Analyzer using Python &amp; Streamlit
Resume Analyzer

This is a simple web application that analyzes resumes and provides a skill-based score.
It helps users understand which skills are present and which are missing in their resume.

---

🚀 Features

- Extracts text from PDF resumes
- Matches skills with predefined keywords
- Calculates resume score
- Displays found and missing skills
- Simple and interactive UI

---

🛠️ Technologies Used

- Python
- Streamlit
- pdfplumber

---

▶️ How to Run

1. Install required libraries
   pip install streamlit pdfplumber

2. Run the application
   streamlit run app.py

3. Upload your resume (PDF)

---

📊 Output

- Resume Score
- Skills Found
- Missing Skills

---

📌 Project Description

This project was built as a beginner-friendly AI-based application to analyze resumes.
It demonstrates skills in Python, file handling, and building web apps using Streamlit.
